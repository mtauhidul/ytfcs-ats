# YTFCS-ATS User Guide - Operational Manual

## Quick Start

### Initial Setup
1. **Login**: Use magic link or password authentication
2. **Access Dashboard**: Navigate to main dashboard after login
3. **Setup Organization Data**: Create tags, categories, and stages first
4. **Start Managing**: Import candidates and create jobs

### System Navigation
- **Sidebar Navigation**: All main features accessible from left sidebar
- **Breadcrumb Trail**: Shows current location and navigation path
- **Search**: Use search bars in each section for filtering data
- **Mobile Responsive**: Interface adapts to different screen sizes

## Core Features - Step-by-Step Instructions

### 1. Jobs Management
**Location**: Click "Jobs List" in sidebar

#### Create a New Job:
1. **Access Job Creation**:
   - Click "Jobs List" in the sidebar
   - Click "Create Job" button (top-right)

2. **Fill Required Information**:
   - **Job Title**: Enter descriptive title (required)
   - **Department**: Enter department name
   - **Location**: Enter job location
   - **Salary Range**: Enter salary range
   - **Employment Type**: Select from dropdown (Full-time, Part-time, Contract, Internship, Temporary)
   - **Status**: Select job status from available statuses
   - **Job Description**: Enter detailed description
   - **Requirements**: Enter comma-separated requirements

3. **Organization** (Optional):
   - **Category**: Select from available categories or leave blank
   - **Tags**: Select multiple tags from available options

4. **Save**: Click "Create Job" button

#### Manage Existing Jobs:
1. **View Jobs**: Jobs display in both table view (desktop) and card view (mobile/tablet)
2. **Search**: Use search box to filter by title, location, department, requirements, etc.
3. **Select Multiple**: Use checkboxes to select multiple jobs
4. **Bulk Actions**: Click "Actions" button when jobs are selected:
   - Change status of selected jobs
   - Add tags to selected jobs
   - Delete selected jobs
5. **Edit Individual Job**: Click on job card or use table actions to open job details
6. **Job Details Modal**: 
   - **Details Tab**: Edit all job information
   - **Tags Tab**: Manage job tags and category
   - **History Tab**: View change history (automatically tracked)
7. **Delete Job**: Use delete button in job details modal

### 2. Candidate Import & Management
**Location**: Click "Candidate Import" in sidebar

#### Upload Resume Files:
1. **File Upload**:
   - Drag & drop PDF, DOC, or DOCX files to upload zone
   - Or click "Browse" to select files
   - Maximum file size: 10MB
   - Only PDF, DOC, DOCX formats accepted

2. **Resume Parsing**:
   - Click "Parse Resume" after upload
   - System extracts: name, email, phone, experience, education, skills
   - Review and edit extracted information as needed

3. **Job Scoring** (Optional):
   - Select a job from "Score Against Job Posting" dropdown
   - Click "Score Resume" to get match percentage
   - Review scoring details

4. **Save Candidate**:
   - Review all parsed/entered information
   - Click "Save Candidate" to add to database

#### Email Import:
1. **Access Email Tab**: Click "Email Import" tab in import page
2. **Configure**: Set up email automation for resume processing
3. **Monitor**: View processing status and imported candidates

### 3. Candidates Database
**Location**: Click "Candidates" in sidebar

#### View and Search:
1. **Candidates List**: View all candidates in data table
2. **Search**: Use search box to filter by name, email, skills, or any field
3. **Bulk Selection**: Use checkboxes to select multiple candidates
4. **Bulk Actions**: Actions button appears when candidates selected

#### Manage Individual Candidates:
1. **View Profile**: Click candidate name to open detailed profile
2. **Edit Information**: Modify candidate details in profile modal
3. **Manage Files**: Download resume files, upload additional documents
4. **Add Notes**: Add interview notes or comments
5. **Update Stage**: Change candidate's current stage
6. **Assign Tags**: Add or remove tags
7. **Rating**: Set candidate rating
8. **View History**: Track all changes made to candidate

### 4. Application Workflow (Kanban Board)
**Location**: Click "Workflow" in sidebar

#### Using the Kanban Board:
1. **View Stages**: Candidates organized in columns by stage
2. **Drag & Drop**: Click and drag candidate cards between stages
3. **Auto-Save**: Stage changes save automatically
4. **Search**: Filter candidates using search box
5. **Candidate Cards**: Show name, tags, and rating
6. **Auto-Scroll**: Board scrolls during drag operations for better UX

### 5. Communication Tools
**Location**: Click "Communication" in sidebar

#### Email Templates:
1. **Create Templates**: Add reusable email templates
2. **Template Variables**: Use {{candidateName}}, {{jobTitle}}, {{companyName}}
3. **Template Types**: Organize by type (interview, rejection, offer, etc.)

#### Messaging:
1. **Send Emails**: Send emails to candidates using templates or custom content
2. **Message History**: Track all sent communications
3. **Status Tracking**: Monitor sent/delivered/read status

#### Settings:
1. **Email Variables**: Configure company-wide email variables
2. **Company Settings**: Set company name and default values

### 6. Team Collaboration

#### Team Management:
1. **View Team**: See all team members and their roles
2. **Invite Members**: Send email invitations to new team members
3. **Assign Candidates**: Assign candidates to specific team members
4. **Role Management**: Manage team member roles and permissions

#### Feedback System:
1. **Submit Feedback**: Rate candidates and provide detailed feedback
2. **Feedback Forms**: Include strengths, weaknesses, and hire/reject recommendations
3. **View All Feedback**: See consolidated feedback from all team members
4. **Feedback History**: Track feedback over time

#### Categories Management:

#### Stages Management:

### 8. Email Monitoring

## Troubleshooting

### Common Issues:
1. **File Upload Problems**:
   - Check file format (PDF, DOC, DOCX only)
   - Verify file size under 10MB
   - Try different browser if issues persist

2. **Search Not Responding**:
   - Check spelling and try simpler keywords
   - Clear search and try again
   - Refresh page if needed

3. **Authentication Issues**:
   - Check email for magic link (including spam folder)
   - Verify password is correct
   - Clear browser cache and cookies

4. **Data Not Saving**:
   - Check internet connection
   - Ensure all required fields are filled
   - Try saving again after page refresh

### Contact Support:
Document error messages and steps to reproduce issues, then contact system administrator.
   - Verify breadcrumb shows current location
   - Click on breadcrumb items to navigate back

## Troubleshooting Guide

### Common Issues and Solutions:

1. **Page Not Loading**:
   - Refresh the browser
   - Check internet connection
   - Clear browser cache

2. **File Upload Fails**:
   - Check file format (must be PDF, DOC, DOCX)
   - Check file size (must be under limit)
   - Try different file

3. **Data Not Updating**:
   - Wait a few seconds for Firebase sync
   - Refresh the page
   - Check internet connection

4. **Drag and Drop Not Working**:
   - Make sure you're clicking and holding the card
   - Drag slowly to target column
   - Release mouse button to drop

5. **Search Not Working**:
   - Check spelling
   - Try partial search terms
   - Clear filters that might interfere

6. **Charts Not Displaying**:
   - Refresh the page
   - Check if there's data to display
   - Wait for data to load
